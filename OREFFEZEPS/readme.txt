 While taking a short ride in Bowser's helicopter, Mario has some aerial trouble and falls into a strange place.

FEATURES:

    27 new spell and attack animations
    16 new items +3 new animations
    1 new weapon animation
    Mario no longer shoots fire orbs...

v2.0 ADDITIONS / FIXES:

    2 new spell animations for Mario
    less HP and lower stats for bosses
    regular monsters give more XP
    freebies 3 times more common
    items have more practical uses

The most outstanding features of this hack were done within the sprite and animation script editors.
